System Overview

This project implements a robust digital logic system for real-time temperature monitoring. It continuously evaluates incoming sensor data against a configurable threshold, utilizing sophisticated debouncing logic to filter out transient glitches and prevent false alarms.

Architecture Details

⏱️

Synchronous Logic

Fully synchronized to the system clock (clk) with active-low reset (rst_n) for predictable state transitions.

🛡️

Glitch Filtering

A parameterized DEBOUNCE_LIMIT ensures alerts are only triggered during sustained abnormal conditions.

🚨

Hardware Alerts

Drives buzzer_on and display_alert output signals directly into the physical alerting mechanisms.
